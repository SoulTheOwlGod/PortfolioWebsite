XQ Competencies

Below are the five XQ competencies with short descriptions and example entries showing how I demonstrated each through projects or activities.

Related Projects
FK
Foundational Knowledge

Understanding core ideas and facts

Strong grounding in web fundamentals (HTML/CSS/JS), accessibility, and performance practices.

Example

Built the portfolio's responsive layout and accessibility features (semantic markup, keyboard navigation, alt text, and performance optimizations like image sizing and critical CSS).

OT
Original Thinker

Apply learning in creative ways

Designing new patterns and solutions for user interaction and visual storytelling.

Example

Implemented a custom animation system for project previews that reduces CPU usage by using transform-based animations and requestAnimationFrame batching.

GC
Generous Collaborator

Work productively with others

Open communication, clear documentation, and mentorship of contributors.

Example

Documented onboarding steps and style guide for a community project, reducing setup time for new contributors and increasing PR quality.

LL
Learner for Life

Curious and continuous improvement

Iterative learning and sharing of new techniques, languages, and practices.

Example

Kept a learning log and built micro-projects for new libraries (e.g., visualization experiments) and wrote short guides to capture lessons learned.

HC
Healthy Creator

Maintain wellbeing while creating

Creating sustainably with balance, attention to community, and iterative timelines.

Example

Structured projects with realistic milestones, logged time and impact, and built routines that prevented burnout while maintaining steady progress.
